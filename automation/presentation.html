Solar Shield Agent

סגירת תריסים אוטומטית לפי קרינת שמש וטמפרטורה

🎯 מטרה

למנוע חימום יתר של הסלון בימים שמשיים וחמים, תוך שמירה על אור טבעי בימים מעוננים או קרים. ההחלטה מתקבלת בזמן חצות היום ההלכתי.

🔧 אמצעים

Data: קרינה גלובלית (W/m²) מתחנת IMS הקרובה — 2 שעות, 10 דקות intervals.
AI: Gemini מנתח ומחליט.
Action: Tuya HMAC-SHA256 signed API.

🧠 שימוש ב-AI

ה-AI מקבל נתונים גולמיים + כללי החלטה ומחזיר Structured JSON עם החלטה בוליאנית + נימוק בעברית שנשלח למשתמש במייל מעוצב.

11:00←Hebcal חצות←⏳ Wait←IMS Data←🤖 Gemini←Tuya←📧

שמירה על טמפרטורה בריאה לשינת ילדים — לא קר מדי ולא חם מדי. חיסכון באנרגיה כשהאוויר החיצוני מספיק.

Data: Tomorrow.io realtime כל שעה.

קיץ: ON מעל 17°C, OFF מתחת.
חורף: ON מתחת 10°C, OFF מעל 12°C.

🌙 Decision Logic — Kids Sleep Guardian

// Kids Sleep Climate Guardian
// Runs EVERY HOUR between 21:00-07:00
// Gets real-time outdoor temperature from Tomorrow.io API

const outdoorTemp = await getTomorrowIO(latitude, longitude);
const month = new Date().getMonth() + 1;
const isSummer = [5,6,7,8,9,10].includes(month);

// ═══════════════════════════════════════════════
// SUMMER LOGIC (May - October)
// AC is pre-set to COOL mode on the device
// ═══════════════════════════════════════════════
if (isSummer) {
  if (outdoorTemp >= 17) {
    → AC ON   // Still warm outside, cooling needed
  } else {
    → AC OFF  // Natural cooling sufficient, save energy
  }
}

// ═══════════════════════════════════════════════
// WINTER LOGIC (November - April)
// AC is pre-set to HEAT mode on the device
// ═══════════════════════════════════════════════
if (!isSummer) {
  if (outdoorTemp < 10) {
    → AC ON   // Cold outside, heating needed for children
  } else if (outdoorTemp > 12) {
    → AC OFF  // Warm enough, no heating needed
  } else {
    → MAINTAIN // Hysteresis zone (10-12°C): don't toggle
  }
}

// At 07:00: Send night summary email with all decisions made
